# Angebote-Ordner

## Anleitung für Mitarbeiter

In diesem Ordner können Sie das aktuelle Angebots-PDF hochladen.

### Wichtig:
- **Dateiname:** Das PDF muss immer `aktuelle-angebote.pdf` heißen
- **Speicherort:** Laden Sie die Datei direkt in diesen Ordner hoch
- **Format:** Nur PDF-Dateien erlaubt
- **Größe:** Maximal 10 MB empfohlen

### Via Strato FTP hochladen:

1. In Strato anmelden
2. FTP-Zugang öffnen
3. Zum Ordner `/angebote/` navigieren
4. Die alte Datei `aktuelle-angebote.pdf` löschen (falls vorhanden)
5. Die neue Datei als `aktuelle-angebote.pdf` hochladen

### PDF entfernen:

Um die Angebote vorübergehend zu deaktivieren, löschen Sie einfach die Datei `aktuelle-angebote.pdf`.
Die Webseite zeigt dann automatisch einen Platzhalter-Text an.
